# 🌐 Frontend चलाने के लिए Guide

## Quick Start

### 1. Streamlit Install करें (अगर नहीं है)
```bash
python3 -m pip install streamlit
```

या सभी dependencies के साथ:
```bash
python3 -m pip install -r requirements.txt
```

### 2. Frontend Run करें
```bash
streamlit run app.py
```

### 3. Browser में खोलें
जब command run होगी, तो automatically browser खुलेगा या आप manually खोलें:
```
http://localhost:8501
```

## Features

✅ **PDF Upload** - नया PDF upload करें  
✅ **Index Building** - Sidebar से index build करें  
✅ **Chat Interface** - Questions पूछें और answers पाएं  
✅ **Context View** - Retrieved contexts देखें  
✅ **Chat History** - Previous conversations देखें

## Troubleshooting

### Port already in use?
```bash
streamlit run app.py --server.port 8502
```

### Streamlit install नहीं हो रहा?
```bash
python3 -m pip install --upgrade pip
python3 -m pip install streamlit
```
